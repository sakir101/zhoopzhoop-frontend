# 🚗 ZhoopZhoop: Car Service Management System

**ZhoopZhoop** is a modern web-based car servicing management system designed to streamline workshop operations. It enables efficient tracking of customer vehicles, service processes, technician activities, and business analytics.
---

## 👥 User Roles
- **Admin**
- **Manager**
- **Advisor**
- **Technician**

---

## 🔑 Key Features

### 🔧 Admin
- Manage appointments based on customer preferences  
- Create and verify customer & vehicle records, and employee profiles  
- Handle full service workflow: **Concern → Inspection → Service → Estimate**  
- Automatically generate and send invoices  
- Analyze sales and generate reports for selected time ranges  

### 📊 Manager
- Manage customer records (with verification)  
- Create estimates and work orders  
- Access a dedicated dashboard  

### 🧑‍💼 Advisor
- Monitor technician progress  
- Notify customers upon service completion  
- Track feedback and additional service needs  
- Assign technicians based on work orders  
- View invoices and sales reports

### 🛠️ Technician
- Access a personal dashboard  
- Update work progress in real-time  
- Communicate with other technicians  
- Filter tasks by **completed**, **incomplete**, and **in-progress** status

## 🔐 Security
- Role-based authentication and authorization  
- Secure login with email and password  
- JWT-based session management using cookies  
- Encrypted passwords in the database  
- Prevention of duplicate accounts (email/contact)  

---

## ⚡ System Design & Performance
- Follows **MVC architecture** for clean and scalable code  
- Optimized for performance and maintainability  

---

## 🛠️ Tech Stack

### Frontend
- Next.js (App Router)  
- TypeScript  
- Redux  
- Tailwind CSS  
- Axios

### Backend
- Node.js  
- Express.js  
- TypeScript  
- Prisma ORM  
- Multer  
- Nodemailer  

### Database
- PostgreSQL
